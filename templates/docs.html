📖 Smart City DSL Documentation

Welcome to the advanced procedural and declarative logic engine. Use natural language commands to control the smart city.

✨ Procedural Examples

toggle-every-3-sec.smart
loop:
Turn on floor1
wait 3 sec
Turn off floor1
wait 3 sec
end loop
fast-blinking-alarm.smart
loop:
break when gas is safe
Turn on mosque
wait 500ms
Turn off mosque
wait 500ms
end loop
conditional-logic.smart
loop:
if light is dark:
Turn on street lights
end if
wait 1000ms
end loop

✨ Declarative Examples

basic-rules.smart
Turn on mosque lights from 18:00 to 23:00.
Switch off castle lights when light is bright.
Keep floor1 always on when gas is safe.
// Zone Dependencies (Chain Rules)
Turn on floor1 when floor2 is turned on.

🔤 Vocabulary Reference

Operations
Turn on Turn off Keep wait X sec wait X ms
Blocks
loop: end loop if [cond]: end if break when
Zones
floor1 floor2 castle mosque street
Sensors
gas safe gas danger light dark light bright
